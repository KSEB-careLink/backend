## 🧠 CareLink Backend

**환자-보호자 기억 유도 앱**의 백엔드 서비스
---
### 📦 기술 스택
| 항목  | 기술                             |
|-------|----------------------------------|
| 서버  | Node.js + Express                |
| 인증  | Firebase Auth + Firestore        |
| 저장소 | Firebase Storage                 |
| 예약알림 | FastAPI + FCM                     |
| 분석용 DB | MySQL (AI 학습/통계용)             |

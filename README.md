# 🧠 CareLink Backend

**환자-보호자 기억 유도 앱**의 백엔드 서비스입니다.  
Firebase 기반 사용자 인증과 Firestore를 중심으로, 회상 정보, 알림, 위치 확인 기능을 제공합니다.

---

## 📦 기술 스택

| 항목 | 기술 |
|------|------|
| 서버 | Node.js + Express |
| 인증 / DB | Firebase Auth + Firestore |
| 파일 저장 | Firebase Storage |
| 예약 알림 | FastAPI (Python) + FCM |
| 데이터 분석 | MySQL (AI 학습용 로그 저장용) |
| 테스트 | Postman |
